快速开始

1. 克隆项目

git clone https://github.com/fuhuiwu/puqee.git
cd puqee

2. 安装依赖

pip install -r requirements.txt

3. 启动ChatBot

Web界面（推荐）：

python main.py --mode http

然后在浏览器中访问：http://localhost:8000/chat-ui

命令行模式：

python main.py --debug
